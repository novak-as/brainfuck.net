# brainfuck.net

##Usage:

  `compiler -f filename [-m memory_size] [-n max_nested_depth]`

##Example:

`compiler -f test.bf -m 50`

#####Input:

* test.bf

   `++++++++++[>+++++++>++++++++++>+++<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.`

#####Compiled program output:
	`Hello World!`
